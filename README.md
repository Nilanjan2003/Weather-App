# 🌤️ Weather App

A modern, responsive Weather Application built using **HTML**, **CSS**, and **JavaScript**. The application fetches real-time weather information and a 5-day forecast using the **OpenWeatherMap API**. It also provides city autocomplete suggestions for a better user experience.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌍 City autocomplete suggestions
- 🌡️ Real-time temperature
- ☁️ Weather condition with icons
- 🤗 Feels-like temperature
- 💧 Humidity information
- 🌬️ Wind speed details
- 📅 5-Day weather forecast
- 🕒 Local date & time based on city timezone
- 📱 Fully responsive design
- 🎨 Modern glassmorphism UI with animated background

---

## 🛠️ Technologies Used

- HTML5

- OpenWeatherMap API

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/weather-app.git
```

2. Open the project folder

```bash
cd weather-app
```

3. Open `index.html` in your browser.

---

## 🔑 API Setup

This project uses the **OpenWeatherMap API**.

1. Create a free account at:
   https://openweathermap.org/api

2. Generate your API Key.

3. Replace the existing API key inside `script.js`

```javascript
const API_KEY = "YOUR_API_KEY";
```
